# MSG Закупки

Система управления закупками металлопроката для МеталлСтройГрупп.

## Возможности

- 📄 Загрузка заявок (Excel, PDF, изображения, текст)
- ⚖️ Автоматический расчёт веса по ГОСТ
- 📧 Рассылка запросов поставщикам
- 📊 Сбор и сравнение цен
- 📈 Формирование сводки лучших предложений

## Технологии

- **Frontend**: HTML/CSS/JavaScript (монолитный index.html)
- **Backend**: n8n workflows (хостинг на Railway)
- **Database**: Supabase (через n8n workflows)
- **Email**: Resend API (через n8n workflows)

## Использование

Откройте [MSG Закупки](https://lamblackout.github.io/msgnn-calculator/) в браузере.

### Основной процесс:
1. **Загрузите заявку** (Excel, PDF, фото или текст)
   - Поддержка нескольких позиций в одной заявке
   - Автоматическое распознавание типов металла

2. **Проверьте распознанные позиции**
   - Редактируйте размеры, марки стали, количество
   - Автоматический пересчёт веса по ГОСТ

3. **Сформируйте запрос поставщикам**
   - Скачайте Excel для проверки данных
   - Выберите категорию поставщиков:
     - 📍 Нижний Новгород
     - 🏙️ Москва и регионы
     - 🏭 Заводы

4. **Получите сводку лучших предложений**
   - Автоматический сбор и сравнение цен
   - Отслеживание активных запросов
   - Скачивание итоговой сводки в Excel
