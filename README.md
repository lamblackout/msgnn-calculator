# MSG Закупки

Система управления закупками металлопроката для МеталлСтройГрупп.

## Возможности

- 📄 Загрузка заявок (Excel, PDF, изображения, текст)
- ⚖️ Автоматический расчёт веса по ГОСТ
- 📧 Рассылка запросов поставщикам
- 📊 Сбор и сравнение цен
- 📈 Формирование сводки лучших предложений

## Технологии

- Frontend: HTML/CSS/JavaScript
- Backend: n8n workflows
- Database: Supabase
- Email: Resend API

## Использование

Откройте [MSG Закупки](https://lamblackout.github.io/msgnn-calculator/) в браузере.

1. Загрузите заявку (Excel, PDF, фото или текст)
2. Проверьте распознанные позиции
3. Отправьте запрос поставщикам
4. Получите сводку лучших предложений
